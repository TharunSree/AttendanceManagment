Enter Marks

Marks Management System

Enter and manage student marks using comprehensive marking schemes with multiple assessment criteria. Support for bulk operations, grade calculations, and result publication.

Understanding the Marking System
How Marking Schemes Work

Each course uses predefined marking schemes that include various assessment criteria:

Common Assessment Types
  • Internal Assessments: Regular tests and quizzes
  • Assignments: Projects and homework
  • Class Participation: Active engagement scoring
  • Practical Work: Lab sessions and experiments
Major Evaluations
  • Mid-term Examinations: Semester midpoint tests
  • Final Examinations: Comprehensive end-of-term tests
  • Project Work: Extended research or creative projects
  • Presentations: Oral and visual presentations
Grade Calculation
  • Each criterion has a defined maximum marks value
  • System automatically calculates total marks and percentages
  • Grade assignments based on institutional grading scale
  • Weightage distribution according to academic policy
Step-by-Step Marks Entry
  1. Go to Faculty Tools → Enter Marks
  2. Select the class group from your assigned classes
  3. Choose the subject you teach
  4. Select the assessment criterion (Internal, Assignment, etc.)
  5. Enter marks for each student:
    • Type marks directly in the input fields
    • Use tab to move between students quickly
    • System validates marks against maximum values
    • Add comments for individual students if needed
  6. Review all entries for accuracy
  7. Save the marks to store permanently
  8. System automatically calculates totals and grades
Validation Rules
  • Marks cannot exceed the maximum value for each criterion
  • Negative marks are not allowed
  • Decimal marks are supported up to 2 decimal places
  • Blank entries are treated as not yet assessed
Importing Marks via CSV

For large classes, use the bulk import feature to upload marks efficiently:

  1. Navigate to Faculty Tools → Bulk Marks Import
  2. Select class, subject, and assessment criterion
  3. Download the CSV template file
  4. Fill in the template with student marks:
    • Student ID and Name columns are pre-filled
    • Enter marks in the designated column
    • Add optional comments in the notes column
  5. Upload the completed CSV file
  6. Review the import preview for errors
  7. Confirm to import all marks
Bulk Import Benefits
  • Faster data entry for large classes
  • Reduced manual typing errors
  • Consistent data formatting
  • Easy import from external tools
Import Precautions
  • Always download the latest template
  • Verify student names and IDs
  • Check data format requirements
  • Review preview before confirming

Extra Classes Management

Extra Classes Overview

Schedule and manage additional classes beyond regular timetable to complete curriculum, provide exam preparation, conduct makeup sessions, or offer remedial support to students.

Scheduling Extra Classes
Step-by-Step Scheduling
  1. Navigate to Faculty Tools → Schedule Extra Class
  2. Select the class group from your assigned classes
  3. Choose the subject you want to teach
  4. Pick an available date (must be in the future)
  5. Select from available time slots
  6. Add class details:
    • Purpose of the extra class (makeup, exam prep, etc.)
    • Topics to be covered
    • Learning objectives
    • Any special requirements or materials
  7. Add optional notes or agenda for students
  8. Submit the request for approval
Scheduling Guidelines
  • Extra classes should be scheduled at least 24 hours in advance
  • Avoid scheduling during other subjects' regular time slots
  • Consider student availability and break times
  • Check for conflicts with examinations or special events
Different Extra Class Categories

  • Makeup Classes: Replace cancelled regular classes

What Happens After Scheduling

  4. Approval Notification: You receive confirmation of approval
  5. Student Notifications: Automatic notifications sent to all students via:

  6. Timetable Integration: Extra class appears in everyone's schedules
